## Installation

### Config
Not available. Please google "react-native install" for hours of rewarding adventures.

### Development, once Your Tooling is Setup
+ Clone this repo.
+ ```cd RN-catalog/```
+ ```npm install```
+ ```react-native run-ios```
+ OR
+ ```react-native run-android```
